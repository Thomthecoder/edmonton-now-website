
# Edmonton Now – Community Website

A multi-page community website built for the Edmonton neighbourhood league.  
Developed as a final project for **COMP1017 – Web Design Fundamentals I** at NAIT.

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox, custom layouts)
- Google Fonts (Commissioner, Open Sans, Oswald)

## 📄 Pages
- **index.html** – Home page with blog posts and community gardens section
- **about.html** – About page with offerings and 2024 events
- **member.html** – Membership page with contact and donation form

## ✨ Features
- Responsive multi-page layout
- Navigation with hover effects
- Banner images with overlay text
- Flexbox card layouts
- HTML form with input validation
- Consistent color branding (#254C65)

COMP1017 – Web Design Fundamentals I
